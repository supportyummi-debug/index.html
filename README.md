# YUMMI

## Bringing Homemade Food Back Home

YUMMI is a startup idea that connects licensed home cooks with people looking for authentic homemade meals in Qatar.

## Features
- Browse homemade meals
- Order from licensed home cooks
- Delivery or pickup
- Secure payments
- Ratings and reviews

## Built With
- HTML5
- Tailwind CSS
- GitHub Pages
